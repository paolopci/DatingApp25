# Technology Stack

## Frontend
- **Framework:** Angular 21 (v21.0.4)
- **Styling:** TailwindCSS con DaisyUI
- **Build Tool:** Angular CLI
- **Testing:** Vitest

## Backend
- **Framework:** .NET 9 Web API (ASP.NET Core)
- **ORM:** Entity Framework Core 9
- **Autenticazione:** JSON Web Tokens (JWT) con Microsoft.AspNetCore.Authentication.JwtBearer
- **Testing:** xUnit con FluentAssertions

## Database
- **Engine:** SQL Server
- **Strumenti:** Migrazioni di Entity Framework Core

## Infrastruttura e Altro
- **Editor:** Visual Studio / VS Code / Google Antigravity
- **Version Control:** Git
